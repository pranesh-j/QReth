## Try it for yourself :)

Clone the project

```bash
  git clone https://github.com/pranesh-j/QReth
```

Go to the project directory

```bash
  cd QReth
```

Install dependencies

```bash
  pip install PyQt5
```

```bash
  pip install qrcode
```
```bash
  pip install web3
```

```bash
  pip install pillow
```
## Run the code
```bash
  python gui.py
```
